# Estudos sobre Ansible

Instalação padrão do Ansible

	yum install python3-pip
	pip3 install --upgrade pip 
	pip3 --version
	pip3 install ansible
	ansible --version
	pip install ansible --upgrade

Remoção do ansible
	
	pip3 uninstall ansible

Instalação por yum
	
	yum install ansible
	yum install epel-release
	yum update
